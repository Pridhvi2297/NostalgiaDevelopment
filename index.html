Our Story

Game Development

This project is designed to highlight similar features that many users found enjoyable in various Mario 2D platformers from the late 1990s and early 2000s. Nostalgia Development developed a game that roped in familiar elements of 2D platformers but also included familiar characters that would bring forth nostalgia for users.

How to run our game:

Before running the game, it requires the following packages:

  • Python
  • PyGame
  • Game Files

Our game can be downloaded as a .zip file containing all the necessary files to run it on user's computers. Please follow the steps below to run the game:

  • >Step 1: Access our GitHub Repository found above

  • >Step 2: Click on the code button and download .zip

  • >Step 4: Extract the files from .zip folder

  • >Step 5: Download Python onto users computer here: https://www.python.org/downloads/

  • >Step 6: Download Visual Studio Code (VSC) onto users computer here: https://code.visualstudio.com/download

  • >Step 7: Navigate to the main.py file using VSC following this path: src/code/levels/main.py

  • >Step 8: Select the 'play/run' button on the top right of VSC

Be aware, if you are attempting to run the game on a Mac, you may be required to install Homebrew and PyGame. Follow the links below to install:
